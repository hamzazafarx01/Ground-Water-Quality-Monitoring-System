# Pin Mapping

| Module | ESP32 Pin | Purpose |

| TDS Sensor | GPIO 35 | Measures dissolved solids in water |
| Turbidity Sensor | GPIO 34 | Measures water cloudiness |
| DS18B20 Temperature Sensor | GPIO 4 | Measures water temperature |
| Ultrasonic Sensor Trigger | GPIO 26 | Sends ultrasonic pulse |
| Ultrasonic Sensor Echo | GPIO 27 | Receives reflected pulse |
| Buzzer | GPIO 25 | Gives alert when water is unsafe |
| LCD SDA | GPIO 21 | I2C data line |
| LCD SCL | GPIO 22 | I2C clock line |
